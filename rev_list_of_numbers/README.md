# Reverse a List of Numbers

A Python program that reverses a list of numbers entered by the user without using built-in reverse methods or slicing.

## Features
- Accepts a number of elements and the list of numbers
- Reverses the list using a loop
- Prints the reversed list in a single line, space-separated

## Concepts Used
- Lists and list iteration
- For-loops with custom range
- User input and type conversion (`int`)
- Appending to lists
- Unpacking lists for printing

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py
