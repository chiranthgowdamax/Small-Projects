# Count Positive, Negative, and Zero Numbers

A Python program that counts how many positive, negative, and zero values are present in a list of numbers entered by the user.

## Features
- Accepts multiple numbers as input
- Counts positive, negative, and zero values separately
- Uses a function to keep logic clean and reusable

## Concepts Used
- Functions and return values
- Lists and list iteration
- Conditional statements
- User input and string splitting
- Type conversion (`int`)

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py
