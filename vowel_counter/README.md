# Vowel Counter

A simple Python program that counts the number of vowels in a given word.

## Features
- Accepts a single word as input
- Counts vowels correctly regardless of letter case
- Uses basic looping and conditional logic

## Concepts Used
- Strings and string iteration
- For-loops
- Conditional statements (`if`)
- User input
- Case conversion (`lower()`)

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py
