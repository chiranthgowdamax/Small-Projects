# Find the Largest Number in a List

A Python program that finds the largest number from a list of numbers entered by the user.

## Features
- Accepts multiple numbers from user input
- Finds the largest number using a simple function
- Handles lists of any size greater than zero

## Concepts Used
- Functions and return values
- Lists and list iteration
- Conditional statements
- User input and type conversion (`int`)
- Basic comparison operators

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py
