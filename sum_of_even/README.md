# Sum of Even Numbers

A simple Python program that calculates the sum of all even numbers from 1 up to a given number.

## Features
- Accepts a number from user input
- Calculates the sum of even numbers only
- Handles small and edge-case inputs safely

## Concepts Used
- Functions
- Conditional statements
- For-loops with step values
- User input and type conversion
- Basic arithmetic operations

## How to Run
Assumption: This project is a single-file Python script named `main.py`.
1. Open a terminal in the project directory.
2. Run:
```bash
python main.py
